# Bloomio

**ドメイン**: bloomio.net  
**コンセプト**: 花にまつわる「言葉」で魅せる、文学的・教養的な花のウェブサイト

## 🌸 プロジェクト概要

Bloomioは従来の花サイトとは一線を画す、**言葉と物語**に特化した花のデータベースです。

### 差別化ポイント

| 一般的な花サイト | Bloomio |
|------------|---------|
| 育て方ガイド | 📖 文学的引用（シェイクスピア、北原白秋） |
| きれいな写真 | 📚 歴史的エピソード（クレオパトラ、薔薇戦争） |
| ガーデニング情報 | 🌏 文化的背景（多国比較） |
| Instagram的 | 💡 うんちく・トリビア |

## 🎯 ターゲット

- **メイン**: 30〜50歳、教養層、読書好き
- **サブ**: 20〜30歳、深い投稿をしたいSNSユーザー
- **興味**: 文学、歴史、文化、教養

## 🚀 クイックスタート

### 1. 依存関係インストール

```bash
cd /root/legacybrands/bloomio/app
npm install
```

### 2. 環境変数設定

`.env.local` ファイルを作成：

```bash
NEXT_PUBLIC_SUPABASE_URL=your-project-url.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

### 3. 開発サーバー起動

```bash
npm run dev
# → http://localhost:3001
```

### 4. ビルド確認

```bash
npm run build
npm start
```
